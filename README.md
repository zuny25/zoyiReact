JOYI React assigment
===

### Requirements
 * Node.js
 * npm

## Getting Started

### Install

```sh
$ npm install --save redux-saga
```

### Development

#### 1. Run api proxy server
```sh
$ npm run api
```

#### 2. Run development frontend
```sh
$ npm run start
```

### Build
```sh
$ npm run build
```

### Run production frontend

#### 1. Run api proxy server
```sh
$ npm run api
```

#### 2. Run development frontend
```sh
$ npm run build:serve
```

### Test
```sh
$ npm run test
```
